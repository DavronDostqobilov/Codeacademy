# Command-line interface

user interface va Command-line interface birgalikda taqqoslab ko`rib ketiladi

pwd -- joriy manzilni ko`rsatadi
print the current working directory

ls -- turgan joyimizdagi folder(papka) va fayllar ro'yxatini chiqarib beradi
list

cd- joriy turgan directoryni o`zgartirish
change directory
cd Desktop/ --directoryni  o`zgartirdik nisbiy manzil bo`ldi chunki chunki directoryimiz ichida Desktop bor edi

mkdir -- folder yaratish

touch -- file yaratish

rm -- file o`chirish

rm -r -- papka o`chirish

rm -f -- papka va file ikkovi uchun ham ishlatdi qaysi biri kelsa ham o`chiradi
